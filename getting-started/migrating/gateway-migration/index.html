Migrating Gateways

For instructions on adding gateways to The Things Stack using the CLI or Console, see Adding Gateways.

When using the Semtech UDP Packet Forwarder, make sure to update the server_address in the gateway configuration settings to the address of the Gateway Server (e.g. my-tts-network.nam1.cloud.thethings.industries).

Once your gateways are migrated, data will be routed to The Things Stack.

Note: If you are within range of The Things Network, data might still end up in The Things Network Stack V2. If this occurs, consider disabling the devices in The Things Network Stack V2 by deleting session keys, or completely deleting the application.
