Frequency Plans

This reference contains Frequency Plans that are officially supported by The Things Stack. This list is populated from the lorawan-frequency-plans Github repository.

Deployments may also have custom frequency plans configured.

Europe 863-870 MHz (SF12 for RX2)

ID Base Frequency Description
EU_863_870 868 Default frequency plan for Europe

Europe 863-870 MHz (SF9 for RX2 - recommended)

ID Base Frequency Description
EU_863_870_TTN 868 TTN Community Network frequency plan for Europe, using SF9 for RX2

United States 902-928 MHz, FSB 1

ID Base Frequency Description
US_902_928_FSB_1 915 Default frequency plan for the United States and Canada, using sub-band 1

United States 902-928 MHz, FSB 2 (used by TTN)

ID Base Frequency Description
US_902_928_FSB_2 915 TTN Community Network frequency plan for the United States and Canada, using sub-band 2

Australia 915-928 MHz, FSB 1

ID Base Frequency Description
AU_915_928_FSB_1 915 Default frequency plan for Australia, using sub-band 1

Australia 915-928 MHz, FSB 2 (used by TTN)

ID Base Frequency Description
AU_915_928_FSB_2 915 TTN Community Network frequency plan for Australia, using sub-band 2

Australia 915-928 MHz, FSB 6

ID Base Frequency Description
AU_915_928_FSB_6 915 Frequency plan for Australia, using sub-band 6, which overlaps with Asia 923-925 MHz

China 470-510 MHz, FSB 11

ID Base Frequency Description
CN_470_510_FSB_11 470 TTN Community Network frequency plan for China, using sub-band 11

Asia 920-923 MHz

ID Base Frequency Description
AS_920_923 915 TTN Community Network frequency plan for Asian countries, using frequencies ≤ 923 MHz

Asia 920-923 MHz with LBT

ID Base Frequency Description
AS_920_923_LBT 915 TTN Community Network frequency plan for Asian countries, using frequencies ≤ 923 MHz with listen-before-talk

Asia 923 MHz with only default channels

ID Base Frequency Description
AS_923 915 TTN Community Network frequency plan for Asian countries, using default frequencies of the AS923 band. This is added only for compatibility. Preferrably use the 920-923 or 923-925 variants

Asia 923-925 MHz

ID Base Frequency Description
AS_923_925 915 TTN Community Network frequency plan for Asian countries, using frequencies ≥ 923 MHz

Asia 923-925 MHz with LBT

ID Base Frequency Description
AS_923_925_LBT 915 TTN Community Network frequency plan for Asian countries, using frequencies ≥ 923 MHz with listen-before-talk

Asia 923-925 MHz (used by TTN Australia)

ID Base Frequency Description
AS_923_925_TTN_AU 915 TTN Community Network frequency plan for Asia 923-925 MHz in Australia

South Korea 920-923 MHz

ID Base Frequency Description
KR_920_923_TTN 915 TTN Community Network frequency plan for South Korea

India 865-867 MHz

ID Base Frequency Description
IN_865_867 868 Default frequency plan for India

Russia 864-870 MHz

ID Base Frequency Description
RU_864_870_TTN 868 TTN Community Network frequency plan for Russia

LoRa 2.4 GHz with 3 channels draft 2

ID Base Frequency Description
ISM_2400_3CH_DRAFT2 2450 LoRa 2.4 GHz plan with 3 channels for global use draft 2
